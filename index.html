LPL-processor

Processing sketch for converting regular videos into light painting videos.

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Probably doesn't require any previous experience with code. It's very simple.

  • Download the Processing IDE from processing.org
  • Download the LPL-processor code (on the left)
  • Put your movie into lpl_processor/data/
  • In the code, edit the moviePath, to match the name of your movie file.
  • Run the script, and wait until it finishes.
  • The composited frames will be in output/. Use the Processing > Tools > Movie Maker utility to create the final movie.
